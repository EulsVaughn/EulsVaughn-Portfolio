Development of Welfare Radar System with Selected Service-Oriented Architecture-Based Assets

Our capstone project, titled "Development of Welfare Radar System with Selected Service-Oriented Architecture-Based Assets,". This project involves creating a system designed to improve the efficiency and effectiveness of welfare services. By utilizing a service-oriented architecture, the system integrates various assets to provide a comprehensive and user-friendly solution for managing and delivering welfare services. This approach ensures that different components can communicate seamlessly, offering a robust framework for enhancing welfare service delivery.

Smart Giga Arena Registration Application

The "Smart Giga Arena Registration Application" was a project developed during my internship at Dark League Studios. This application was designed to streamline the event registration process, providing a user-friendly experience for both customers and staff. By addressing issues such as internet connectivity, input errors, human errors, and empty inputs, the application significantly sped up the registration process and improved overall efficiency.
